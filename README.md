# דף נחיתה — קורס ספרות (דמו)

דף נחיתה אחד (one-page) למכירת קורס ספרות אונליין. עברית מלאה (RTL), רספונסיבי, עיצוב כהה ופרמיום.

## איך צופים בדף
שלוש דרכים:
1. **בפאנל התצוגה המקדמית** של Claude Code (כבר פתוח).
2. **פתיחה ישירה** של `index.html` בדפדפן (דאבל-קליק על הקובץ).
3. **שרת מקומי** (מומלץ): מהתיקייה הזו הרץ
   ```bash
   python3 -m http.server 8123
   ```
   ואז גלוש ל-http://localhost:8123

## איך משנים תוכן  ✏️
**כל התוכן נמצא במקום אחד:** בראש הקובץ `script.js`, באובייקט `CONFIG`.
שם משנים: שם המותג, שם הספר, מספר וואטסאפ, אינסטגרם, מחירים, גלריה, סילבוס, המלצות ושאלות נפוצות.
אחרי עריכה — שומרים את הקובץ ומרעננים את הדף.

| מה לשנות | איפה ב-`CONFIG` |
|---|---|
| שם המותג / הספר | `brandName`, `instructorName` |
| וואטסאפ ואינסטגרם | `whatsappNumber`, `instagramUser` |
| מחירים | `priceBasic`, `pricePro` … |
| תמונות וסרטונים | מערך `gallery` |
| נושאי הקורס | מערך `curriculum` |
| המלצות | מערך `reviews` |
| שאלות נפוצות | מערך `faq` |

### גלריה — תמונות וסרטונים
במערך `gallery` כל פריט הוא תמונה או סרטון:
```js
// תמונה
{ type: "image", src: "assets/cut1.jpg", caption: "פייד קלאסי" }
// סרטון (פותח נגן בלחיצה)
{ type: "video", src: "assets/thumb.jpg", caption: "תהליך מלא",
  embed: "https://www.youtube.com/embed/VIDEO_ID" }
```
תמונות מומלץ לשים בתיקייה `assets/` ולהפנות אליהן (למשל `assets/cut1.jpg`).

> כרגע התמונות הן ממלאי-מקום אקראיים (picsum). מחליפים אותן בעבודות אמיתיות.

## מה עוד צריך לחבר (השלב הבא)
- **תשלום + שליחת הקורס למייל** — דרך Gumroad / Lemon Squeezy. הקישורים ייכנסו ל-`checkoutBasic` / `checkoutPro`.
- **טופס לידים אמיתי** — כרגע מציג הודעת תודה בלבד (דמו). נחבר ל-Formspree או לוואטסאפ.
- **דומיין ואחסון** — העלאה חינמית ל-Vercel / Netlify / GitHub Pages.

## קבצים
- `index.html` — מבנה הדף
- `styles.css` — עיצוב (כל הצבעים במשתנים בראש הקובץ, קל לשנות גוון)
- `script.js` — תוכן (CONFIG) + אינטראקטיביות
