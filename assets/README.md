# 📁 תיקיית מדיה — תמונות וסרטונים

כאן שמים את כל התמונות והסרטונים של העבודות.
פשוט גוררים לכאן את הקבצים — **אין צורך בשמות מיוחדים**, אני אחבר הכל לדף.

## מה כדאי לשים
- **1 תמונת רקע להירו** — לרוחב, הכי איכותית
- **1 תמונת פורטרט של שנהב** — לאורך
- **6–9 תמונות עבודות** — לאורך עדיף
- **2–3 סרטונים קצרים** — או קישורי אינסטגרם/יוטיוב במקום קבצים

## טיפים
- איכות: תמונות חדות ומוארות. צילומי טלפון טובים בהחלט.
- כיוון: גלריה ופורטרט — לאורך. הירו — לרוחב.
- סרטונים: אם הם כבר באינסטגרם — עדיף לשלוח קישור (נטען מהר וחוסך מקום).
